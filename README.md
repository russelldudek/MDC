## Hero technology
The Promise Compiler is a self-contained cinematic renderer with no production CDN dependency. It uses a native WebGPU raymarching path when supported, a matched WebGL2 raymarching path as the GPU fallback, and a designed Canvas 2D crystalline fallback when neither GPU context is available. WebGPU shader compilation is validated before that backend can own the canvas. A stateful semantic SVG object is always present beneath the GPU atmosphere, so device or renderer failure cannot collapse the hero into blank space.

The five scenarios are framed as operating decisions: renewal decision, start-readiness gap, scope change request, agent evidence failure, and forecast/cash mismatch. Each changes the object geometry, fault mode, operating decision, decision owner, and required evidence as one state transaction. Motion runs once and settles; replay is deliberate. Mobile retains the same meaning through a recomposed, lightweight scene rather than a squeezed desktop dashboard.
