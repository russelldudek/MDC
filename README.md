## Hero technology
The Engagement Load Path is a self-contained GPU simulation split into a native renderer core, fallback renderers, and a small bootstrap layer. It uses native WebGPU when the browser and secure context support it, WebGL2 as the GPU fallback, and a visually equivalent Canvas 2D fallback when a GPU context is unavailable. Narrow mobile devices and reduced-motion users receive a lightweight semantic composition rather than a squeezed desktop renderer.

Each scenario changes the physical load deformation, six phase states, margin-and-cash foundation, founder intervention, operating decision, named owner, and required evidence as one state transaction. Motion runs once and settles; replay is deliberate.
