# Chief Operating Officer - AI Adoption Candidate Vision

Independent candidate campaign by Russell Dudek for the confidential AI adoption and consulting firm represented by MDC Executive Search.

## Routes
- `index.html` - candidate vision, continuous Operating Attractor Field, and operating stress test
- `resume.html` - fluid screen view and exactly two-page printable resume
- `cover-letter.html` - fluid screen view and exactly one-page printable cover letter
- `interview-brief.html` - executive interview thesis
- `120-day-plan.html` - entry plan
- `role-alignment.html` - one-page alignment brief

## Hero technology
The Operating Attractor Field is a self-contained Canvas 2D simulation with no production CDN dependency. It shows volatile founder-generated demand moving through scope, capacity, evidence, and cash controls until the work stabilizes into accountable delivery lanes.

The field runs as a role-specific operating cycle: demand enters, the operating system stabilizes it, the result holds long enough to read, and the field reopens for the next wave of demand. Rendering pauses when the hero is offscreen. `prefers-reduced-motion` resolves immediately to the stable semantic end state and does not loop.

## Screen and print documents
Browser document views use a continuous editorial composition with readable type and responsive spacing. Print remains isolated in `styles.css`, preserving exact approved pagination. Native PDF downloads remain available from each document toolbar.

## Integrity
The client identity is not disclosed in the posting. This campaign does not guess publicly. Company-specific operating ideas are labeled as hypotheses for discovery. Career claims are bounded to verified candidate evidence.
